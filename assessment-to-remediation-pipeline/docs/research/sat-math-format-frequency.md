# SAT Math Format And Frequency Parameters

## Purpose

This document defines item-format parameters for the original one-hour diagnostic.

## Full SAT Math Format Reference

Official College Board pages describe SAT Math as:

- two modules,
- 44 total questions,
- 70 total minutes,
- questions from all four Math domains in each test module,
- questions arranged from easier to harder within each module,
- mostly 4-option multiple choice,
- approximately 25% student-produced response.

## MVP Format

| Format | Count | Percent |
| --- | ---: | ---: |
| Four-option multiple choice | 27 | 75% |
| Numeric student-produced response | 9 | 25% |
| Total | 36 | 100% |

## Item-Authoring Rules

- Each MCQ item has 4 choices.
- Each MCQ item has exactly one correct choice.
- Each incorrect choice must be plausible and tied to a misconception or procedural error.
- Each MCQ item includes feedback for each distractor.
- Each numeric SPR item includes accepted-answer rules.
- Numeric SPR answer rules should support exact values, equivalent fractions/decimals, ranges, or tolerance where appropriate.
- V1 does not use essay-style constructed response.
