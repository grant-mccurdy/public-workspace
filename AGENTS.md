# AGENTS.md

## Project Purpose

This repository is the public portfolio and presentation layer for my career transition.

Its purpose is to demonstrate technical, analytical, educational, and systems-building capability through public-facing projects, case studies, dashboards, demos, and GitHub Pages assets.

The repository should help employers quickly understand:

- what problems I solve,
- what systems I build,
- how I think,
- what tools I use,
- and where I can create value.

## Public / Private Boundary

This repository is public-facing.

Allowed:
- portfolio projects
- case studies
- dashboards
- synthetic datasets
- public-safe screenshots
- technical documentation
- GitHub Pages assets

Not allowed:
- student data
- parent data
- personnel information
- internal school documents
- application trackers
- recruiter/contact databases
- private correspondence
- API keys
- credentials
- tokens
- confidential employer information

When in doubt, keep the material in the private repository.

## Portfolio Positioning

Position the repository owner as:

- mathematics and statistics professional
- educational systems builder
- assessment and analytics practitioner
- AI-assisted workflow developer
- Canvas/LMS systems builder
- technical problem solver

Avoid:
- grievance language
- confidential details
- unsupported claims
- AI hype without artifacts
